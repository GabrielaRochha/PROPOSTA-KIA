# Proposta Rápida — CD Tecnologia (Opção A)
Site estático para preencher e imprimir propostas rapidamente. Sem backend, 100% GitHub Pages.

## Estrutura
- `index.html` — página principal (form + preview A4 + imprimir)
- `style.css` — estilos de impressão e utilidades
- `script.js` — sincronização dos campos com o preview

## Como usar
1. Abra `index.html` no navegador.
2. Preencha os campos à esquerda.
3. Clique em **Imprimir / PDF** para gerar o arquivo.

## Hospedar no GitHub Pages
1. Crie um repositório no GitHub (ex.: `proposta-rapida`).
2. Envie esses arquivos para a raiz do repositório.
3. Em **Settings → Pages**, selecione Branch `main` e `/ (root)` e salve.
4. O site ficará disponível em: `https://SEU-USUARIO.github.io/proposta-rapida/`.

> Dica: personalize o cabeçalho (logo, nome, contatos) direto no `index.html`.
